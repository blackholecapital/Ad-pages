=== INVOICE COMPOSER CONTRACT ===
FILE: apps/product-shell/src/features/payme/InvoiceComposer.tsx

Required input fields:
  invoiceNumber   string   auto-generated INV-YYYYMMDD-XXXX; editable
  amount          number   dollar amount (0.01 step, min 0); required
  subject         string   invoice subject / description; optional
  notes           string   textarea; payment terms, instructions; optional

Required output actions:
  Generate Payment Link
    — enabled when invoiceNumber + amount are non-empty
    — builds URL: {origin}/{slug}/payme?inv=&amt=&subj=
    — displays link in invoiceLinkBox with Copy button
    — resolved by GET /api/invoice-link?inv=&amt=&subj= (Cloudflare Function)

  Download PDF
    — enabled when invoiceNumber + amount are non-empty
    — opens print-ready HTML document in new window
    — HTML built client-side from buildPrintHtml() (reuses invoiceBuilders escaping pattern)
    — triggers window.print() after 350 ms render delay
    — output: browser print-to-PDF suitable for email/text attachment
