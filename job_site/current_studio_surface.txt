=== PAGE SCOPE (CURRENT) ===
Studio does NOT have a hard-coded page picker limiting which pages can be decorated.
The Studio editor (DesktopPremiumStudio) operates on a layout object (PremiumShellLayout) passed in memory.
No page manifest or allowlist currently restricts which published pages Studio can target.
External Studio URL (https://studio.xyz-labs.xyz/) is referenced in AdminPage but content of that service is outside this repo.

=== CLOUDFLARE FUNCTIONS RELEVANT TO STUDIO OUTPUT ===

apps/product-shell/functions/api/published-manifest.js — serves tenant manifest from R2
apps/product-shell/functions/_lib/runtime-compiler.js — compiles runtime page from manifest+layout
R2 bucket DEMO_BUCKET (binding name) contains Studio/demo bundles at key shape: json/<slug>/site.json
