=== AUTH GATE MECHANISM ===
Component: apps/product-shell/src/components/gate/RequireGate.tsx
Type: React Router layout route (pathless, element: <RequireGate />)
Pattern: wraps child routes; renders <Outlet /> when access is granted, renders LoginModal gate when not

=== ACCESS CONDITIONS ===
Access granted when ANY of:
  1. bypassEnabled = true  (Demo bypass — local operator flow)
  2. connected = true AND signed = true  (wallet connected + message signed)

Access denied (gate shown) when:
  bypassEnabled = false AND (connected = false OR signed = false)

=== GATE STATE ===
State source: local useState within RequireGate (not global DemoGateContext)
Persistence: session only (state resets on page reload)
State fields:
  connected: boolean     — wallet connected via WalletConnectButton
  signed: boolean        — message signed via SignMessageButton
  bypassEnabled: boolean — demo bypass toggled via DemoBypassButton

=== GATE UI (when unauthenticated) ===
Component: LoginModal (title: "Access Gate")
Contents:
  - GateStatusPill: shows current step label (Wallet required / Wallet connected / Access verified / Demo bypass enabled)
  - Descriptive copy: "Connect wallet and sign a message to continue."
  - WalletConnectButton: sets connected = true on success
  - SignMessageButton: enabled only when connected; sets signed = true on success
  - DemoBypassButton: toggles bypassEnabled for local operator flows

=== GATED ROUTES (router.tsx — RequireGate group) ===
admin                         → AdminPage
studio                        → StudioPage
:slug/admin                   → AdminPage
:slug/studio                  → StudioPage
:designation/:slug/admin      → AdminPage
:designation/:slug/studio     → StudioPage

=== TRIGGER PATH ===
User clicks Admin nav link → navigates to /admin or /:slug/admin
  → matched by RequireGate layout route
  → if unauthenticated: RequireGate renders LoginModal gate (AdminPage NOT rendered)
  → user completes wallet connect + sign (or enables bypass)
  → RequireGate renders <Outlet /> → AdminPage renders
